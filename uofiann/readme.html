Here is a practical example of an artificial neural net (ANN) which runs over images from the U of I chest Xray collection. The sample size of a few thousand Xrays is too smal to build a seriously robust ANN, however the code is provided as an example of how an ANN can be built quickly and efficiently. The training data are available online- look up in the readme for the repository. Notice at present I am putting up examples of NLP by a sort of brute force method of simply looking for strings. THIS DOES NOT ACTUALLY SORT THE REPORTS IN A MEANINGGFUL WAY!!!! THIS DOES NOT SORT THE REPOTS IN A WAY SUITABLE FOR BUILDING AN ARTIFICIAL NEURAL NETWORK! More elegant solutions exist with additional libraries.
